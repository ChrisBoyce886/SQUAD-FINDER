About

Users will have the ability to create an event in the application with all of the details necessary such as total number of members, location, etc. Once an event is created, the website will pull information from the Google Maps/Yelp APIs to show the information on the locations. People will then be able to search for events in their area (based on a specific radius) and sign up for the event.
